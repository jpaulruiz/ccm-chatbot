How it's built

Proven stack. We already know it.

Next.js + Supabase

Dashboard, auth, Postgres, pgvector, and file storage — all in one service. No Pinecone, no separate S3, no extra infrastructure to manage.

OpenAI + Claude

GPT-4o mini for lighter bots, Claude Haiku for mid-tier, Claude Sonnet for the highest accuracy — swappable via Vercel AI SDK with no code changes.

Shadow DOM widget

One Vite-built script tag. The bot runs in an isolated Shadow DOM — zero CSS conflicts on any WordPress, Webflow, or Shopify site we manage.

Security by default

Each bot gets a scoped embed key locked to one domain. Server-side LLM key management — no API keys ever exposed to the browser or the nursery.
